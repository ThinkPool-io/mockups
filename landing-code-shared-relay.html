Why it’s different

Scaling agents isn’t the same as working together.

Running several agents at once is fan-out: one person, more hands. Thinkpool is lateral. Two people share one Ensemble, both able to watch and steer, while every agent terminal can see the others and hand work across the room.

⌘Multiple providers, one room

Choose a model for the job.

Run Claude Code or Codex. Claude lanes can use an Anthropic-compatible endpoint directly, including a compatible local endpoint, or reach other models through OpenRouter.

⬡Ensemble

Agents work across terminals.

Every agent terminal can see the others and delegate a defined slice to them. Work stays visible in the Ensemble instead of disappearing into a private agent tree.

↔Pair

Two people steer together.

Both people can watch any terminal, step in, and redirect the work. One person can hand a running lane to the other without losing context.
