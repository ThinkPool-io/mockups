Discipline — the judgment agents lack
diagnose-firstverify

After a failed fix, stop and re-diagnose with fresh evidence — never ship the same guess twice.

root-cause-fixverify

Bug showing up in 2+ places? Read the whole data path and fix the root once, not each symptom.

verify-claimsverify

Never say "done" or cite a number without runtime evidence you produced. Green build ≠ it runs.

verify-in-prodverify

Reported broken again? Pull the real failing artifact before the next patch — not a synthetic repro.

use-existing-scriptsverify

Check for a tool that already does it before reinventing curl/poll/grep mechanics inline.

reland-revertedverify

How it works
Three steps, zero config

Skills are just markdown + scripts in the open agentskills.io format. Claude reads them automatically and invokes them by description — or by their trigger word.

01

Install

Run npx thinkpool-skillpack add — or clone into .claude/skills/.

02

Nothing to wire

Claude auto-loads each skill at session start and reaches for it when it fits the task.

03

Or say the word

Hotword skills fire on trigger: heist stripe.com, forge add caching.
