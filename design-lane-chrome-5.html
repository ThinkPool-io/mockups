Five ways to make Design feel like an Ensemble lane.

The current build makes Design a primary terminal tab, then repeats its identity in the viewer header. These directions put it where it belongs, reduce the top chrome, and turn the bottom panel into a contextual tool instead of a permanent instruction block.

Why it sits outside now: the Ensemble rail lives inside an active terminal pane; Design replaces that pane.
01

Nested rail

Safest IA fix. Design becomes the active Ensemble sibling; one slim identity bar owns page name and mode.

My read

Start from 01. It fixes the lane placement with the smallest conceptual jump and removes the redundant identity. Borrow 05’s responsive behavior for desktop: bottom composer on mobile, side inspector when there is width. Keep 03 as the “canvas maximum” alternative if we want Design to feel less like a terminal and more like a shared object.
